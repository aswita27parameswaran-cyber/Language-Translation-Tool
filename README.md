# Language Translation Tool

## Description

Language Translation Tool developed using Python, Tkinter, and Deep Translator. The application allows users to enter text, select source and target languages, and instantly receive translated output through a user-friendly graphical interface.

## Features

* Translate text between multiple languages
* Simple GUI using Tkinter
* Fast translation using Deep Translator
* Clear and easy-to-use interface

## Technologies Used

* Python
* Tkinter
* Deep Translator

## How to Run

1. Install Python 3.x
2. Install Deep Translator:
   pip install deep-translator
3. Run:
   python translator.py

## Output

The application translates text from the selected source language to the target language and displays the result instantly.
